# IronOS

## О проекте
Это проект операционной системы с открытым исходным кодом, который я начал **13 июля 2026 года**. Архитектура построена на принципах **микроядра**. Я не пытаюсь конкурировать с гигантами индустрии — я просто создаю то, что мне нравится. 

Для запуска вам потребуются базовые утилиты: **GCC, NASM, Make**. Сборка и запуск выполняются простой командой:
```bash
make run
```
